# 2026-04-12

## Implementation Cycle

### Card Art Fix
Replaced the MiniMax API with deterministic CSS gradient art in `/api/generate-card-image/route.ts`. The MiniMax API (`api.minimax.io`) is unreachable from the server (connection hangs with no response), so image generation was hanging indefinitely. Now: try MiniMax with 8s timeout, fall back to `getCardArtFallback()` which returns unique SVG gradient art per card.

### GameLobby Match Detection Fix
`handlePlayVsBot` was checking `row.player1Id === playerId` only. If `startSinglePlayerMatch` assigns the player as `player2`, the match would never be found. Now checks both `player1Id` and `player2Id`, and polling window doubled from 2s to 4s.
